Bachelor of Science in Information Technology (B.Sc. IT)

Empowering students with modern IT skills for a digital future.

Course Overview

B.Sc. in Information Technology is a 3-year undergraduate program focusing on software development, networking, data structures, and IT fundamentals. This course prepares students for careers in the fast-evolving field of Information Technology by providing both theoretical and practical knowledge.

Key Subjects Covered

  • Programming in C/C++
  • Data Structures & Algorithms
  • Database Management Systems
  • Computer Networks
  • Operating Systems
  • Web Development (HTML, CSS, JS)
  • Software Engineering
  • Cloud Computing
  • Mobile App Development
  • Cyber Security Basics

Career Opportunities

After completing B.Sc. IT, students can explore careers such as:

Software Developer
Network Administrator
Database Manager
Web Developer
IT Support Specialist
Cyber Security Analyst
